# sugar_management
 
間食で摂取した糖質を見える化することを目的とします。  
食べた物（間食）を数値として見ることで、自身の健康、特に糖尿病予防を考えるきっかけになればと思います。

***
## 機能一覧

ユーザー登録  
食べた物（間食）の数量と日付を登録  
日付毎に摂取した総糖質を確認できる  
一日に何を食べたかの確認ができる
